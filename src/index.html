Производительность

Sodium + Iris дают +300% FPS по сравнению с ванильным клиентом. Установи их для Fabric.

Оптимизация памяти

Не выделяй Java больше 8 ГБ — это не ускоряет игру. Оптимально 2–4 ГБ для большинства сборок.

Быстрый мир

Используй seed-ы для поиска идеального спавна. Лучшие ресурсы — в seed с деревней рядом со спавном.

Java 21+

Minecraft 1.20.5+ требует Java 21. Для старых версий (1.16 и ниже) лучше Java 8.

Резервные копии

Регулярно делай бэкапы папки saves. Один краш может удалить прогресс сотен часов.

F3 меню

Нажми F3 в игре чтобы увидеть FPS, координаты, биом и загрузку чанков в реальном времени.
